# 근무자표 자동완성 프로그램
## :mega: 소개
>군대에서 불침번 및 위병소 근무의 자동화를 위한 **근무자표 자동완성 프로그램** 입니다.  
>개발기간은 2023.5.29 ~ 2023.6.15 입니다.

## 1. 구현 계획 및 내용
###  [고려할 요소]</br>
  1. 시간대별 가중치 부여(**비선호하는 시간대 공평하게 분배**)</br>
  2. **출타**(휴가, 외박) 고려</br>
